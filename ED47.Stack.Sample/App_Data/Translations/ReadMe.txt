In this folder add you translations XML files.

Sample:

<?xml version="1.0" encoding="utf-8"?>
<translations lang="en">
  <sample>
    <title>Sample</title>
    <message>Hello world!</message>
  </sample>
</translations>

Don't forget to set your translation file's language in the root element!
Don't forget to change the name of the second element with something of you choice.

Name your translations files with the following convention: rootKey.language.xml
Specifying the language in the name if its the default language is optional

Example: admin.fr.xml
